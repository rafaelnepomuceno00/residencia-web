# Residência Médica — Planilha Web (TypeScript)

Aplicação web estática (Vite + React + TypeScript) que lê a planilha `Planilha_Residencia_Medica.xlsx` **no navegador** e mostra os dados em abas com tabela, busca e ordenação.

## Rodar localmente

```bash
npm install
npm run dev
```

## Como funciona

- A planilha padrão fica em `public/Planilha_Residencia_Medica.xlsx` e é carregada automaticamente.
- Você pode usar **Importar XLSX…** para abrir outra planilha (não envia nada para servidor).

## Deploy no GitHub Pages

1. Suba este projeto para um repositório no GitHub (branch `main`).
2. No GitHub: **Settings → Pages → Build and deployment → Source: GitHub Actions**.
3. Faça um push na `main`. O workflow `Deploy to GitHub Pages` vai publicar.

> Observação: este projeto usa `base: './'` no Vite para funcionar em qualquer subpasta do Pages.

# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
